# `personalization`

`personalization` 개체에는 사용자에 따라 사용자 경험을 조정할 수 있는 몇 가지 속성이 있습니다. 이 기능은 사용자별로 표시된 콘텐츠를 사용자 지정할 수 있는 원동력이므로 Adobe Target 또는 Adobe Journey Optimizer 구현에서 특히 중요합니다.

자세한 내용은 [개인화된 콘텐츠를 렌더링](../../personalization/rendering-personalization-content.md)하는 방법에 대한 설명서를 참조하십시오.
