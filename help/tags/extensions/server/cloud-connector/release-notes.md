# Adobe Experience Platform Cloud Connector 확장 기능 릴리스 정보

>[!NOTE]
>
>Adobe Experience Platform Launch는 Adobe Experience Platform의 데이터 수집 기술로 새롭게 브랜딩되었습니다. 그 결과로 제품 설명서 전반에서 몇 가지 용어 변경이 있었습니다. 용어 변경에 대한 통합 참고 자료는 다음 [문서](../../../term-updates.md)를 참조하십시오.

## 2023년 1월 17일

v1.0.1

* Body Raw 텍스트 영역에 유효한 JSON을 붙여넣을 때 JSON이 아닌 문자열로 저장되는 문제를 해결했습니다.
* GET 또는 HEAD 요청에 Body를 설정하지 마십시오.
* 5KB보다 큰 응답을 저장하면 규칙 실행이 실패하는 버그를 수정했습니다.
