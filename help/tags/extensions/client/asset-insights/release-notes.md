# AEM 자산 통찰력 릴리스 노트

## 2020년 9월 9일

**AEM 자산 통찰력 확장 1.0.3**

**기능**

- 이 확장은 AEM 자산 통찰력과 함께 사용됩니다.
- 특히 ***pageTracker*** 프로세스와 페이지 코드를 대체합니다.
- 이 확장이 구성 및 활성화되면 AEM 자산 노출 횟수 및 클릭 지표를 Adobe Analytics으로 전송하며 해당 지표는 자동으로 AEM의 자산 통찰력 보고서로 다시 가져옵니다.

**버그 수정**

- 없음
