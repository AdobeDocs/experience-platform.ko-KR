# Adobe Experience Platform 웹 SDK 태그 확장 개요

Adobe Experience Platform Web SDK 태그 확장은 Adobe Experience Platform Edge Network을 통해 웹 속성에서 Adobe Experience Cloud으로 데이터를 전송합니다. Experience Platform Web SDK 확장을 사용하면 데이터를 플랫폼으로 스트리밍하고 ID와 옵트인을 동기화하며 컨텍스트 데이터를 자동으로 수집할 수 있습니다.

이 태그 확장을 설정하고 배포하여 데이터 수집을 시작하는 방법에 대해 알아보려면 [시작하기](getting-started.md)를 참조하십시오.

한 번의 호출로 [!DNL Web SDK], [!DNL Edge Network], [!DNL Experience Platform] 및 [!DNL Analytics]&#x200B;(으)로 데이터를 전송하여 아래 비디오를 통해 Adobe Experience Platform [!DNL Audience Manager] 및 [!DNL Target]의 작동 방식을 확인하십시오.
