# 작업 유형

이 페이지에는 태그 규칙 내에서 지원되는 모든 작업이 나열됩니다. 태그 규칙 내에서 작업을 만들거나 편집하려면 다음을 수행하십시오.

1. Adobe ID 자격 증명을 사용하여 [experience.adobe.com](https://experience.adobe.com)에 로그인합니다.
1. **[!UICONTROL Data Collection]** > **[!UICONTROL Tags]**(으)로 이동합니다.
1. 원하는 태그 속성을 선택합니다.
1. **[!UICONTROL Rules]**(으)로 이동한 다음 원하는 규칙을 선택합니다.
1. [!UICONTROL Actions]에서 기존 작업을 선택하거나 작업을 만듭니다.
1. [!UICONTROL Extension] 드롭다운 필드를 **[!UICONTROL Adobe Experience Platform Web SDK]**(으)로 설정한 다음 원하는 [!UICONTROL Action Type]을(를) 선택합니다.

웹 SDK 태그 확장은 다음 작업을 제공합니다.

* [**[!UICONTROL Apply propositions]**](apply-propositions.md): 지표를 증가시키지 않고 단일 페이지 애플리케이션에서 제안을 렌더링합니다.
* [**[!UICONTROL Apply response]**](apply-response.md): Edge Network의 응답을 기반으로 작업을 수행합니다.
* [**[!UICONTROL Evaluate rulesets]**](evaluate-rulesets.md): 규칙 집합 평가를 수동으로 트리거합니다.
* [**[!UICONTROL Get Media Analytics tracker]**](get-media-analytics-tracker.md): 기존 Media API를 창 개체로 내보냅니다.
* [**[!UICONTROL Redirect with identity]**](redirect-with-identity.md): 소유하고 있는 도메인 간에 방문자 식별자를 공유할 수 있습니다.
* [**[!UICONTROL Send event]**](send-event.md): Edge Network에 이벤트 데이터를 보냅니다.
* [**[!UICONTROL Send media event]**](send-media-event.md): Edge Network에 미디어 데이터를 보냅니다.
* [**[!UICONTROL Set consent]**](set-consent.md): 방문자에 대해 원하는 동의를 설정합니다.
* [**[!UICONTROL Update variable]**](update-variable.md): [변수](../data-element-types.md#variable) 데이터 요소를 수정합니다.
