# BrightCove Video Tracking 확장 릴리스 노트

## 2020년 7월 6일

### BrightCove 비디오 추적 1.1.0

#### 기능

* 한 페이지에서 여러 비디오를 추적하는 패치

## 2020년 6월 14일

### BrightCove 비디오 추적 1.0.3

#### 기능

* 임베드된 `<videojs>` 태그를 사용하는 Brightcove 플레이어에서 시작, 일시 중지, 다시 시작, 완료 등과 같은 비디오 이벤트 및 사용자 지정 가능한 큐 포인트를 쉽게 구성할 수 있도록 해주는 Adobe Analytics용 사용자 지정 링크 추적 기반 솔루션입니다.
