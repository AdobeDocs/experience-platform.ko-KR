## 사용자 인터페이스 업데이트 {#ux}

Adobe Experience Platform은 경험을 향상하고 다른 과의 통합을 위해 도메인 및 헤더 막대를 업데이트하고 있습니다 [!DNL Experience Cloud] 응용 프로그램.

- 조직 간 또는 다른 애플리케이션으로 쉽게 전환할 수 있습니다
- 도움말 메뉴의 주요 문서 및 컨텍스트 관련 설명서를 비롯한 사용자 도움말이 개선되었습니다
- 에 대한 피드백 제공 기능 [!DNL Experience Platform] 및 파일 지원 티켓

새로운 경험에서 점차 벗어나고 있다. 에서 경험을 볼 수 있습니다 [https://experience.adobe.com/platform](https://experience.adobe.com/platform).

## [!DNL Real-Time Customer Profile] {#profile}

Adobe Experience Platform을 사용하면 고객이 브랜드와 상호 작용하는 위치와 시기에 관계없이 고객을 위해 조정되고 일관되며 적절한 경험을 제공할 수 있습니다. 사용 [!DNL Real-Time Customer Profile]를 사용하면 온라인, 오프라인, CRM 및 타사 데이터를 포함하여 여러 채널의 데이터를 결합하는 각 개별 고객을 전체적으로 확인할 수 있습니다. [!DNL Profile] 서로 다른 고객 데이터를 모든 고객 상호 작용을 실행 가능하고 타임스탬프가 지정된 계정을 제공하는 통합 보기에 통합할 수 있습니다.

**새로운 기능**

| 기능 | 설명 |
| -----------| ---------- |
| 신규 프로필 내보내기 지표 | 내보낸 프로필의 총 수와 각 네임스페이스의 프로필 수를 보여주는 프로필 내보내기 작업에 대한 지표를 추가했습니다. |
| 새로운 가시성 통찰력 지표 | 이제 Observability Insights API에는 수집 스트리밍을 위한 다음 지표가 있습니다 [!DNL Profile]: 들어오는 요청 비율, 성공적인 수집 비율, 수집된 레코드 크기입니다. |
| 벌크 GET 끝점 | 에 벌크 GET 종단점을 추가했습니다. [!DNL Real-Time Customer Profile] 단일 API 호출에서 여러 결과를 검색할 수 있도록 설정하는 API입니다. 이제 세그먼트 정의, 세그먼트 작업 및 병합 정책에 대해 최대 100개의 ID를 벌크로 GET 할 수 있습니다. |
| ID로 프로필 찾아보기 | 에서 [!DNL Platform] 이제 UI에서 ID 네임스페이스를 선택하고 ID 값을 제공하여 프로필을 검색할 수 있습니다. |

**버그 수정**

- None.

- None.

## 소스 {#sources}

Adobe Experience Platform은 외부 소스에서 데이터를 수집할 수 있으며 를 사용하여 해당 데이터를 구조화, 레이블 지정 및 향상시킬 수 있습니다 [!DNL Platform] 서비스. Adobe 애플리케이션, 클라우드 기반 스토리지, 타사 소프트웨어 및 CRM 시스템과 같은 다양한 소스에서 데이터를 수집할 수 있습니다.

[!DNL Experience Platform] 는 다양한 데이터 공급자에 대한 소스 연결을 쉽게 설정할 수 있는 RESTful API 및 대화형 UI를 제공합니다. 이러한 소스 연결을 통해 외부 스토리지 시스템 및 CRM 서비스를 인증 및 연결하고, 수집 실행 시간을 설정하고, 데이터 수집 처리량을 관리할 수 있습니다.

**새로운 기능**

| 기능 | 설명 |
| ------- | ----------- |
| 클라우드 스토리지 시스템에 대한 추가 API 및 UI 지원 | 용 새 소스 커넥터 [!DNL Azure File Storage]. |
| 데이터베이스에 대한 추가 API 및 UI 지원 | 용 새 소스 커넥터 [!DNL Azure Data Explorer], IBM DB2 및 [!DNL Oracle DB]. |
| Adobe Audience Manager에 [!DNL Experience Platform] 데이터 공유 | 에 대한 프로비저닝 프로세스 [!DNL Audience Manager] 커넥터가 업데이트되었습니다. [!DNL Audience Manager] 데이터 세트 [!DNL Real-Time Customer Profile] 이제 기본적으로 비활성화되어 있습니다. 프로모션할 데이터 세트를 수동으로 선택할 수 있습니다 [!DNL Profile]. 새 기본 설정은 소급 적용되지 않으며 새 설정에 대한 프로비저닝에만 영향을 줍니다 [!DNL Audience Manager] 커넥터. 자세한 내용은 [데이터 세트 사용 안내서](../../catalog/datasets/user-guide.md). |
