# 임시 스키마

특정 상황에서 다음을 만들어야 할 수 있습니다. [!DNL Experience Data Model] 단일 데이터 세트에서만 사용할 수 있도록 네임스페이스가 지정된 필드가 있는 XDM 스키마. 이를 임시 스키마라고 합니다. 임시 스키마는 CSV 파일 수집 및 특정 유형의 소스 커넥터 생성을 비롯한 여러 데이터 수집 워크플로우에서 사용됩니다. 에서 만드는 방법에 대한 단계를 포함하여 임시 스키마에 대한 자세한 내용을 보려면 [!DNL Schema Registry] API, 다음을 참조하십시오. [임시 스키마 튜토리얼](../tutorials/ad-hoc.md).
