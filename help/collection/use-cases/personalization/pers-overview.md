# Personalization 사용 사례 개요

Adobe Experience Platform 웹 SDK을 통해 기업은 웹 속성에 다양한 개인화 사용 사례를 제공할 수 있습니다. Target, Journey Optimizer 및 Audience Manager과 같은 Adobe 솔루션 간에 데이터 수집, 의사 결정 및 활성화 프로세스를 원활하게 연결할 수 있습니다.

SDK은 클라이언트측, 서버측 및 하이브리드 모델을 비롯한 유연한 아키텍처를 지원하므로 대부분의 조직 사용 사례를 충족하는 개인화된 오퍼를 가져와 렌더링할 수 있습니다.

## 웹 SDK 샘플

이 폴더의 문서 페이지 외에도 Adobe은 참조할 수 있는 샘플 애플리케이션의 저장소를 유지 관리합니다. 다음을 포함한 개인화 사용 사례는 GitHub에서 [웹 SDK 샘플](https://github.com/adobe/alloy-samples/)을 참조하십시오.

* 클라이언트측 개인화
* 서버측 개인화
* 하이브리드 개인화
* 단일 페이지 애플리케이션을 사용하는 Personalization
