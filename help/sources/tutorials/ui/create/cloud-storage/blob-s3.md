# UI에서 [!DNL Azure Blob] 또는 [!DNL Amazon] S3 소스 커넥터 만들기

Adobe Experience Platform의 소스 커넥터는 예약된 기준으로 외부 소스 데이터를 인제스트하는 기능을 제공합니다.

클라우드 스토리지 소스는 다운로드, 형식 지정 또는 업로드할 필요 [!DNL Platform] 없이 고유한 데이터를 가져올 수 있습니다. 인제스트된 데이터는 XDM JSON, XDM 쪽모이 세공식 또는 구분 기호로 형식을 지정할 수 있습니다. 프로세스의 모든 단계는 소스 워크플로우에 통합됩니다. [!DNL Platform] 일괄 처리 [!DNL Azure Blob] 에서 데이터를 가져올 수 [!DNL Amazon S3] 있습니다.

소스 커넥터 사용에 대한 자세한 내용은 [!DNL Azure Blob] 소스 커넥터 안내서를 참조하십시오 [[!DNL Azure Blob] ](./blob.md).

소스 커넥터 사용에 대한 자세한 내용은 [!DNL Amazon S3] 소스 커넥터 안내서를 참조하십시오 [[!DNL Amazon S3] ](./blob.md).
