# UI에서 [!DNL Deltashare] 소스 커넥터 사용 {#use-deltashare-in-the-ui}

>[!CONTEXTUALHELP]
>id="platform_sources_deltashare_schema"
>title="스키마 구조"
>abstract="계속하면 더 이상 스키마를 변경할 수 없으므로 스키마 구조를 검토해야 합니다."

Adobe Experience Platform 사용자 인터페이스에서 [!DNL Deltashare] 소스 커넥터를 사용하는 방법을 알아보려면 이 안내서를 참조하십시오.

**참고:** **다음**&#x200B;을(를) 선택하면 선택한 스키마 구조를 변경할 수 없습니다.

**다음**&#x200B;을(를) 이미 선택하고 스키마 선택 단계를 지나서 이동한 경우 이전 단계로 돌아가면 선택한 스키마를 더 이상 업데이트할 수 없습니다. 스키마를 수정하려면 데이터 흐름 구성 프로세스를 다시 시작하고 초기 단계부터 시작해야 합니다.
