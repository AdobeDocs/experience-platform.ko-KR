# [!DNL Phoenix]

Adobe Experience Platform Sources는 다음과 같은 타사 데이터베이스에서 데이터 수집을 지원합니다. [[!DNL Phoenix]](https://phoenix.apache.org/index.html). 이 문서에서는 [!DNL Phoenix] 계정 스루 [!DNL Flow Service] API 또는 Experience Platform 사용자 인터페이스.

## IP 주소 허용 목록

소스 커넥터로 작업하려면 먼저 IP 주소 목록을 허용 목록에 추가해야 합니다. 지역별 IP 주소를 허용 목록에 추가하지 않으면 소스 사용 시 오류가 발생하거나 성능이 저하될 수 있습니다. 다음을 참조하십시오. [IP 주소 허용 목록](../../ip-address-allow-list.md) 페이지 를 참조하십시오.

아래 설명서는 연결 방법에 대한 정보를 제공합니다 [!DNL Phoenix] API 또는 Experience Platform 인터페이스를 사용하여 검색하려면 다음 작업을 수행하십시오.
