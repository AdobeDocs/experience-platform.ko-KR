# (베타) [!DNL Couchbase] 커넥터

>[!NOTE]
>커넥터의 [!DNL Couchbase] 베타입니다. 베타 [레이블이 지정된 커넥터 사용에 대한 자세한 내용은 소스 개요를](../../home.md#terms-and-conditions) 참조하십시오.

Adobe Experience Platform은 MySQL과 같은 데이터베이스 제공업체를 위한 기본 연결 [!DNL Microsoft]을 제공하므로 이러한 시스템에서 데이터를 가져올 수 [!DNL Azure]있습니다. 관계형, NoSQL 또는 data warehouse을 비롯한 다양한 유형의 타사 데이터베이스가 지원됩니다. 데이터베이스 공급자에 대한 지원에는 다음이 포함됩니다 [!DNL Couchbase].

아래 설명서에서는 API 또는 사용자 인터페이스 [!DNL Couchbase] 를 [!DNL Platform] 사용하여 연결하는 방법에 대해 설명합니다.
