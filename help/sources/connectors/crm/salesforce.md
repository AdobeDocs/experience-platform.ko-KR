# [!DNL Salesforce] 커넥터

Adobe Experience Platform을 사용하면 [!DNL Platform] 서비스를 사용하여 들어오는 데이터를 구조화, 레이블 지정 및 향상시키는 기능을 제공하면서 외부 소스에서 데이터를 수집할 수 있습니다. Adobe 애플리케이션, 클라우드 기반 스토리지, 데이터베이스 등 다양한 소스의 데이터를 인제스트할 수 있습니다.

[!DNL Experience Platform] 타사 CRM 시스템의 데이터 인제스트 지원을 제공합니다. CRM 공급자에 대한 지원에는 다음이 포함됩니다 [!DNL Salesforce].

아래 설명서에서는 API 또는 사용자 인터페이스 [!DNL Salesforce] 를 [!DNL Platform] 사용하여 연결하는 방법에 대해 설명합니다.
