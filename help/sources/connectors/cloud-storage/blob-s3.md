# Azure Blob 및 Amazon S3 커넥터

Adobe Experience Platform은 AWS, [!DNL Google Cloud Platform], 및 [!DNL Azure]. 이러한 시스템의 데이터를 [!DNL Platform].

클라우드 스토리지 소스는 고유한 데이터를 [!DNL Platform] 를 다운로드하거나, 형식을 지정하거나, 업로드할 필요가 없습니다. 수집된 데이터는 XDM JSON, XDM Parquet 또는 구분된 형식으로 지정할 수 있습니다. 프로세스의 모든 단계는 소스 워크플로우에 통합됩니다. [!DNL Platform] 에서 데이터를 가져올 수 있습니다. [!DNL Azure Blob] 및 S3 - 일괄 처리를 지원합니다.

에 대한 자세한 내용은 [!DNL Azure Blob] 커넥터, [[!DNL Azure Blob] 커넥터 개요](./blob.md).

에 대한 자세한 내용은 [!DNL Amazon S3] 커넥터, [[!DNL Amazon S3] 커넥터 개요](./s3.md).
