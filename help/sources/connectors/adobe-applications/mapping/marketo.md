# [!DNL Marketo Engage] 필드 매핑 {#marketo-engage-field-mappings}

>[!CONTEXTUALHELP]
>id="platform_sources_marketo_mapping"
>title="Marketo 소스 필드 매핑"
>abstract="Marketo와 플랫폼 사이에 소스 연결을 설정하려면 플랫폼에 수집되기 전에 Marketo 소스 데이터 필드를 적합한 대상 XDM 필드에 매핑해야 합니다."

아래 표에는 9개 필드 간의 매핑이 포함되어 있습니다 [!DNL Marketo] 데이터 세트 및 해당 XDM(경험 데이터 모델) 필드.

>[!TIP]
>
>모두 [!DNL Marketo] 데이터 세트 제외 `Activities` 지금 지원 `isDeleted`. 기존 데이터 흐름에 자동으로 다음이 포함됩니다. `isDeleted`는 새로 수집된 데이터에 대한 플래그만 수집합니다. 모든 이전 데이터에 플래그를 적용하려면 기존 데이터 흐름을 중지하고 새 매핑을 사용하여 다시 만들어야 합니다. 다음을 제거할 경우 참고 `isDeleted`를 클릭하면 더 이상 기능에 액세스할 수 없습니다. 매핑은 자동으로 채워진 후에도 유지되어야 합니다.

## 활동 {#activities}

다음 [!DNL Marketo] 이제 source에서 추가 표준 활동을 지원합니다. 표준 활동을 사용하려면 를 사용하여 스키마를 업데이트해야 합니다.
