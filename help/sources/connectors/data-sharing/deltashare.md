# [!DNL Deltashare]

[!DNL Deltashare] 소스 커넥터를 사용하여 [!DNL Databricks Delta Sharing] 서버에 대한 보안 연결을 설정하고 Adobe Experience Platform에서 가상 데이터 세트를 만들 수 있습니다. 이 기능을 사용하면 가상 데이터 세트를 쿼리하고 외부 테이블을 검사하고 XDM(Experience Data Model) 스키마에 매핑하여 Experience Platform 애플리케이션 및 서비스에서 활용할 수 있습니다. [!DNL Deltashare]을(를) 통해 데이터 세트의 물리적 수집을 줄일 수 있으므로 중복, 스토리지 비용 및 운영 오버헤드를 줄일 수 있습니다.
