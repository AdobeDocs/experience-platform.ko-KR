# 허용 목록에 추가하다 IP 주소

네트워크 방화벽을 통해 네트워크 액세스 제어를 정의할 수 있습니다. 적절한 소스 IP 범위를 지정하면 데이터 전송 서비스에 대한 트래픽이 흐르도록 할 수 있습니다. 소스 커넥터를 사용하기 전에 다음 IP 주소를 허용 목록에 추가하다에 추가해야 합니다. 영역별 IP 주소를 허용 목록에 추가하다에 추가하지 않으면 소스를 사용할 때 오류나 성능이 저하될 수 있습니다.
