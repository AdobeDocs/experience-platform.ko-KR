# 샌드박스 UI 개요

Adobe Experience Platform에서는 단일 플랫폼 인스턴스를 별도의 격리된 가상 환경으로 분할하는 가상 샌드박스를 제공합니다. 샌드박스 사용자 인터페이스는 [Experience Platform UI](https://platform.adobe.com)에서 찾을 수 있습니다.

Experience Platform UI에 로그인하고 나면 [샌드박스 사용 안내서](user-guide.md)에서 샌드박스 작업 영역에서 다양한 작업을 수행하는 방법에 대한 단계를 계속 진행하십시오.

Sandbox API를 사용하여 다른 작업을 수행하는 방법은 [샌드박스 개발자 안내서](../api/getting-started.md)를 참조하십시오. Experience Platform의 샌드박스에 대한 높은 수준의 개요를 보려면 [개요 설명서](../home.md)를 참조하십시오.
