# Privacy Service 오류 코드 {#privacy-service-error-codes}

이 참조를 사용하여 **Adobe Experience Platform**&#x200B;에서 개인 정보 보호 작업을 제출하거나 모니터링할 때 Privacy Service 작업 결과를 식별하고, 오류를 진단하고, 적절한 다음 단계를 결정합니다. 개인 정보 작업을 만들고, 제출하고, 모니터링하는 방법은 [개인 정보 작업 끝점 안내서](./privacy-jobs.md) 또는 [Privacy Service UI 사용 안내서](../ui/user-guide.md)를 참조하십시오.

Privacy Service 오류 코드는 안정적인 공개 계약입니다. 각 오류 코드는 프로그래밍 방식의 처리 및 운영 워크플로에 사용할 수 있는 오류 또는 완료 상태를 고유하게 식별합니다.

자동화를 구축하거나 워크플로를 모니터링할 때에는 다음 보장이 적용됩니다.

* 오류 코드는 게시되면 안정적입니다.
* 명확성을 개선하기 위해 오류 메시지가 변경될 수 있지만 코드 값은 변경되지 않습니다.
* 새로운 오류 코드는 시간이 지남에 따라 추가될 수 있으며 기존 코드는 다시 사용되지 않습니다.

메시지 텍스트가 아닌 오류 코드를 사용하여 자동화 또는 의사 결정 논리를 구현합니다. 폴링이나 메시지 문자열에 의존하지 않고 개인 정보 작업을 효율적으로 처리하고, 작업 상태를 모니터링하고, 오류를 처리하는 방법에 대한 지침은 [Privacy Service 모범 사례](../best-practices.md)를 참조하세요.

## 오류 응답 형식 {#error-response-format}

Privacy Service이 작업 및 요청 응답에 오류 정보를 반환합니다. 응답은 숫자 오류 코드와 응답 페이로드에 사람이 읽을 수 있는 메시지를 포함한다.

오류 코드는 신뢰할 수 있는 결과를 전달합니다. 이 메시지는 문제 해결을 위한 추가 컨텍스트를 제공합니다.

이 문서에서는 각 오류 코드의 의미와 의도를 설명합니다. 필드 수준 응답 스키마 및 요청 세부 정보는 [Privacy Service API 설명서](https://developer.adobe.com/experience-platform-apis/references/privacy-service/)를 참조하세요.

## 오류 도메인 {#error-domains}

오류 코드는 문제를 보다 신속하게 진단할 수 있도록 기능 도메인별로 그룹화됩니다.

이 문서에 사용되는 도메인은 다음과 같습니다.

* **요청 유효성 검사**: 요청의 형식이 잘못되었거나 잘못된 값이 있습니다. 요청 구조 및 유효성 검사 요구 사항에 대해서는 [개인 정보 작업 끝점 안내서](./privacy-jobs.md)를 참조하십시오.
* **권한 부여 및 프로비저닝**: 조직 또는 사용자에게 필요한 액세스 권한이 없습니다. 역할 기반 권한 요구 사항을 검토하려면 [권한 관리](../permissions.md)를 참조하십시오.
* **ID 및 적용 가능성**: 식별자 또는 네임스페이스를 요청에 적용할 수 없습니다. 지원되는 ID 유형 및 네임스페이스 요구 사항은 [개인 정보 보호 요청에 대한 ID 데이터](../identity-data.md)를 참조하십시오.
* **속도 제한**: 전송 볼륨이 플랫폼 제한을 초과합니다. 이 오류가 발생하면 전송 속도를 줄이고 다시 시도하십시오.
* **데이터 액세스 및 처리**: 시스템이 요청된 데이터에 액세스하거나 처리할 수 없습니다. 일반적인 원인 및 수정 단계는 [문제 해결 안내서](../troubleshooting-guide.md)를 참조하십시오.
* **암호화 및 키 관리**: 필요한 암호화 키를 사용할 수 없습니다. 키 액세스, 구성 및 복구 지침은 [고객 관리 키](../../landing/governance-privacy-security/customer-managed-keys/overview.md)를 참조하세요.
* **작업 실행 상태**: 작업이 전체, 부분적으로 완료되었거나 오류가 발생했습니다. 작업 상태 범주 및 해당 의미에 대한 설명은 [개인 정보 작업 끝점 안내서](./privacy-jobs.md#status-categories)를 참조하십시오.

>[!NOTE]
>
>도메인 할당은 내부 서비스 경계가 아니라 오류 코드의 의도를 반영합니다.

## 오류 코드 참조 {#error-code-reference}

다음 표에는 모든 공개 Privacy Service 오류 코드가 나와 있습니다.

| 오류 코드 | HTTP 상태 | 제목 | 설명 |
| ---------- | ----------- | ----------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------| 
| 1000-400 | 400 | 서식 지정 오류 | 지정한 응용 프로그램에 대한 하나 이상의 데이터 값에 서식 문제가 있습니다. 자세한 내용은 작업 세부 사항을 확인하십시오. |
| 1001-400 | 400 | 권한이 없음 | 조직이 프로비저닝되지 않았습니다. 자세한 내용은 관리자에게 문의하십시오. |
| 1010-400 | 400 | 권한 누락 | 이 작업을 수행하는 데 필요한 권한이 없습니다. 관리자에게 문의하여 액세스 권한을 요청하십시오. |
| 1020-400 | 400 | 업로드 및 보관 실패 | 액세스 데이터를 업로드 및 보관하는 도중 문제가 발생했습니다. 액세스 데이터를 업로드하고 다시 시도하십시오. |
| 1021-400 | 400 | 작업 실패 | 요청에서 만든 하나 이상의 개인 정보 보호 작업이 실패했습니다. 자세한 내용은 실패한 작업의 세부 정보를 검토하십시오. |
| 1022-400 | 400 | 데이터 액세스 실패 | 지정한 데이터에 액세스하는 동안 문제가 발생했습니다. 자세한 내용은 작업 세부 사항을 검토하십시오. |
| 1023-400 | 400 | 데이터 액세스 실패 | 지정된 데이터 세트 ID에 액세스하거나 찾는 동안 문제가 발생했습니다. 입력한 ID가 유효한지 확인한 후 다시 시도하십시오. |
| 1024-400 | 400 | 예기치 않은 오류 | 예기치 않은 오류가 발생했습니다. 자세한 내용은 작업 세부 사항을 검토하십시오. |
| 1030-400 | 400 | 문서 속도 제한 초과 | 작업량이 문서 속도 제한을 초과했습니다. 전송 속도를 줄인 후 다시 시도하십시오. |
| 1040-400 | 400 | 키 암호화 액세스 실패 | 데이터 저장소가 암호화되고 키 액세스가 취소되었으므로 데이터를 처리할 수 없습니다. 고객 키에 대한 액세스를 복원하려면 Key Vault 관리자에게 문의하십시오. |
| 6000-200 | 200 | 성공 | 작업이 완료되었습니다. 작업 세부 사항을 검토하여 처리된 레코드 및 결과를 확인합니다. |
| 6051-200 | 200 | 프로비저닝되지 않음 | 요청한 애플리케이션에 대해 조직이 프로비저닝되지 않습니다. 요청을 적용할 수 없습니다. |
| 6052-200 | 200 | 사용자 ID를 찾을 수 없음 | 일부 사용자 ID를 찾을 수 없으며 알 수 없는 사용자 ID는 이 제품에 적용할 수 없습니다. 사용자 ID가 유효한지 확인한 후 다시 시도하십시오. |
| 6053-200 | 200 | 잘못된 네임스페이스 | 입력한 ID 네임스페이스가 이 애플리케이션에 유효하지 않습니다. 시스템에서 인식하는 네임스페이스를 사용합니다. |
| 6054-200 | 200 | 부분적으로 완료됨 | 적용 가능한 데이터에 대한 작업이 완료되었지만 일부 데이터가 요청에 적용되지 않았습니다. 자세한 내용은 작업 세부 사항을 검토하십시오. |
