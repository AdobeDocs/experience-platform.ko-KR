# 대상을 연결하는 방법

실시간 고객 데이터 플랫폼의 대상에 연결하려면 아래 절차를 따르십시오.

1. 왼쪽 **[!UICONTROL 탐색 막대에서]** 대상으로 이동합니다.
2. [ [[!UICONTROL 카탈로그]](./destinations-workspace.md#catalog) ] 탭에서 연결할 대상을 선택합니다.
3. Connect **[!UICONTROL 대상]** 키를 눌러 연결 대상 마법사를 시작합니다.
4. 특정 대상에 연결하는 방법에 대한 단계별 지침은 대상 페이지를 참조하십시오. 클라우드 [스토리지 대상](../catalog/cloud-storage/workflow.md), [이메일 마케팅 대상](../catalog/email-marketing/overview.md)및 [광고 대상을 참조하십시오](../catalog/advertising/overview.md).

대상을 연결하고 세그먼트를 대상을 활성화하는 방법에 대한 비디오 개요는 아래를 참조하십시오.
