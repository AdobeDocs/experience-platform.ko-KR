# 개인화 대상 {#personalization-destinations}

개인화 플랫폼은 개별 행동을 통해 데이터 및 통찰력을 수집합니다. 이러한 플랫폼은 마케터가 익명의 데이터를 탐색하거나, 이전 활동을 기반으로 잠재 고객에게 제안이나 광고를 제공하거나 유사한 습관을 가진 고객 세그먼트의 일부로서 제공할 수 있도록 지원합니다.
