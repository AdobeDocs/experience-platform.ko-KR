# 데이터 집합

경험 플랫폼 내의 데이터 집합 작업 영역을 사용하면 IMS 조직에서 만든 모든 데이터 집합을 보고 관리하고 새 데이터 집합을 만들 수 있습니다.

왼쪽 탐색 창에서 클릭하여 데이터 세트 작업 영역을 **[!UICONTROL Datasets]** 봅니다. 데이터 집합 작업 영역에는 데이터 집합이 마지막으로 업데이트된 날짜 및 시간뿐만 아니라 이름, 생성됨(날짜 및 시간), 소스, 스키마 및 마지막 일괄 처리 상태를 표시하는 열을 포함한 데이터 집합 목록이 들어 있습니다.

데이터 집합 작업 영역 사용에 대한 자세한 내용은 데이터 집합 [개요를](../../catalog/datasets/overview.md)참조하십시오.
