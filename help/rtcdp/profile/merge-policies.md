# 실시간 고객 데이터 플랫폼에서 정책 병합

Adobe Experience Platform을 기반으로 구축된 [!DNL Real-time Customer Data Platform]을 사용하면 여러 소스의 데이터를 취합하여 각 개별 고객의 전체 상황을 파악할 수 있습니다. 이 데이터를 함께 가져올 때 병합 정책은 데이터의 우선 순위를 지정하는 방법과 통합 보기를 만들기 위해 결합할 데이터를 결정하는 데 사용하는 규칙입니다.[!DNL Platform]

사용자 인터페이스를 통해 새 병합 정책을 만들고 기존 정책을 관리하고 조직에 대한 기본 병합 정책을 설정할 수 있습니다. 데이터 소스에 대한 환경 설정을 지정하고 병합하는 동안 충돌이 발생할 경우 우선 순위가 있는 데이터를 결정할 수도 있습니다.

Adobe Experience Platform 사용자 인터페이스를 사용하여 정책 병합 사용자 안내서](../../profile/ui/merge-policies.md)에 따라 병합 정책을 사용하기 위한 단계별 지침을 찾을 수 있습니다.[
