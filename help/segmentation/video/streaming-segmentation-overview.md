# Experience Platform UI의 스트리밍 세그멘테이션

다음 비디오는 스트리밍 세그멘테이션에 대한 이해를 지원하기 위한 것입니다. 고객 경험의 예를 보여주고, [!DNL Platform] 인터페이스에서 주요 기능을 빠르게 둘러볼 수 있습니다. 스트리밍 세그멘테이션 및 UI에서 스트리밍 세그멘테이션을 수행하는 방법에 대한 자세한 내용은 [스트리밍 세그멘테이션 UI 안내서](../ui/streaming-segmentation.md)를 참조하십시오.

세그멘테이션 API를 사용한 스트리밍 세그멘테이션에 관심이 있는 경우 [스트리밍 세그멘테이션 API 안내서](../api/streaming-segmentation.md)를 참조하십시오.

>[!NOTE]
>
>Experience Platform 사용자 인터페이스는 자주 업데이트되며 이 비디오 레코딩 이후 변경되었을 수 있습니다. 최신 정보는 [스트리밍 세그멘테이션 UI 안내서](../ui/streaming-segmentation.md)를 참조하십시오.
