# 플랫폼 설명서 언어 지원

Adobe Experience Platform 설명서는 여러 언어로 제공됩니다. 언어를 변경하려면 언어 드롭다운을 클릭하고 원하는 언어를 선택합니다.

![image](../images/overview/lang.jpg)

## [!DNL Experience Platform] 설명서

사용 가능한 문서 유형에 대한 자세한 내용은 [플랫폼 설명서 유형 개요를 참조하십시오](overview.md). 특정 [!DNL Platform] 기능 또는 서비스에 대한 자세한 내용을 보려면 [Experience Platform 설명서 랜딩 페이지에서 안내서를 선택합니다](https://experienceleague.corp.adobe.com/docs/experience-platform.html?lang=en).
